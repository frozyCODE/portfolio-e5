💻

Le Projet : Suite HoloNet

Refonte et modernisation des applications internes (Django-D, Yan Solo).

🎯

Problématiques

  • Ergonomie : Tableaux surchargés, navigation fastidieuse (52 cases à cocher).
  • Fiabilité : Processus manuels (Excel), risques de calculs erronés sur des données sensibles.

Les Choix Techniques

  • Refonte UX : Choix du composant USelectMenu (NuxtUI) pour fluidifier radicalement la recherche utilisateur.
  • Automatisation : Connexion API validée via Postman pour supprimer la saisie manuelle.
  • Sécurité Financière : Decimal.js vital pour contourner les imprécisions natives de JavaScript sur le calcul de la TVA.

Bilan & Impact Métier

  • Impact Métier : Gain de temps quotidien crucial pour les comptables et managers.
  • Bilan Technique : Interface moderne, données fiabilisées, risques financiers écartés.
  • Bilan Personnel : Maîtrise du workflow collaboratif (Pull Requests) et de l'écosystème Vue/Nuxt.

Stage 2 : Laboratoire TIMC

L'Organisation (CNRS / UGA)

  • Activité : Unité de recherche publique en imagerie et données médicales.
  • Mon rôle : Stagiaire Développeur Java - Architecture Logicielle & Données.
💻

Le Projet : CardiacTS

Développement d'une application métier sur-mesure dédiée aux chercheurs en imagerie cardiaque.

🎯

Problématique

Concevoir une architecture logicielle capable de respecter le Plan de Gestion des Données (PGD) de la recherche publique tout en gérant des volumes massifs de données médicales.

Solution apportée

Création d'une application Java/JavaFX et d'une infrastructure conteneurisée (Docker) avec un stockage hybride : PostgreSQL pour les métadonnées et MinIO pour les fichiers lourds.

Bilan

Consolidation des compétences en architecture logicielle, Plan de Gestion des Données, utilisation avancée de GitHub et Gestion de Projet.
